# 技术栈

- **Next.js 16 / React 19 / TypeScript 5**：页面路由、SSR/CSR 能力和类型安全基础。
- **Tailwind CSS + Radix UI**：提供响应式布局、明暗主题和可访问的交互组件。
- **Zustand**：集中式管理 UI 状态（语言、校区/服务商筛选、引导弹窗、聚焦站点、跟踪高亮等）。
- **TanStack Query**：统一管理远程数据（站点、服务商），负责缓存、错误处理、刷新与状态派发。
- **ECharts + AMap 插件**：渲染地图与站点分布。
- **Biome / pnpm**：格式化、Lint 与包管理工具链。
